# 📱 Bluetooth Chat Application

This application is a **Bluetooth-based chat app** developed for communication between two Android phones. Users can connect two devices through Bluetooth and send messages to each other in real time.

## ✨ Features

- 🔍 Bluetooth device scanning
- 🔗 Device pairing and connection
- 💬 Real-time messaging between two phones
- 🎨 Simple and user-friendly interface

## 📋 Requirements

- Two Android phones
- Bluetooth enabled on both devices
- App installed on both phones

## 🚀 How to Run the Application

### Step 1: Install the App
Install the APK file on both Android devices.

### Step 2: Turn On Bluetooth
Enable Bluetooth on both phones from device settings.

### Step 3: Open the Application
Launch the application on both devices.

### Step 4: Scan Devices
Click on the **"Scan Devices"** button to search for nearby Bluetooth devices.

### Step 5: Pair Devices
Select the opposite device from the list and pair/connect both phones.

### Step 6: Connect Devices
After pairing, click on the device name on both phones. The application will show a **"Connected"** status.

### Step 7: Start Chatting
Type a message on one device and send it. The message will appear on the connected device instantly.

## 🛠️ Technologies Used

- Android Studio
- Java / Kotlin
- Bluetooth API
